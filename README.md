# Game To Aim Induction: Arcade Classics

A collection of two classic arcade games developed for the **GAME TO AIM** induction. This repository features a minimalist implementation of **Snake** and **Dino Run**, focusing on clean logic and responsive gameplay.

## üéÆ Included Games

### 1. Snake Game

The quintessential classic. Navigate the snake to eat food and grow longer while avoiding walls and your own tail.

* **Objective:** Achieve the highest score possible.
* **Controls:** Arrow Keys or WASD to navigate.

### 2. Dino Game

A fast-paced "infinite runner" inspired by the Chrome Dino game. Jump over obstacles that increase in speed over time.

* **Objective:** Survive as long as possible.
* **Controls:** Spacebar or Up Arrow to jump.
